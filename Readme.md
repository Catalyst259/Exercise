10/6：数组与指针（Part1）
数组定义：int/char/double... array[digit]
数组取值：array[x]
练习：找最大，找最小，求平均，逆序排列，冒泡排序

指针定义：type *p = &x
运算中 p == &x == "x的地址"，但*p == x
*与&互为逆运算，前者将地址映射到其对应值，后者将一个值映射到它对应的地址
&(*p)的值为指针p指向的变量x的地址
*(&x)的值为变量x的值
array在运算中衰变为地址&array[0]，可以给指针变量赋值
指向常量的指针不可修改变量值
常量指针不可修改指向地址
指针（地址变量）可以与整数相加减，比如在数组中，*(array+1) = array[1]
在同一数组或字符串中，指针可以与指针相减，得到一个整型差值
